**COMP 550 | Spring 2026 | M.S. Computer Science**

---

## About the Project

CI Needs is a peer-to-peer student resource network built for the Cal State Channel Islands community. Our mission is to connect students to resources that can help them continue with their education — whether that means finding a meal, a textbook, a laptop, or just knowing that someone in the Dolphin community is ready to help.

We believe that no student should have to pause or abandon their academic journey because of a basic need that the community around them could meet. CI Needs makes it simple to ask for help, offer support, and keep each other moving forward.

| `index.html` | Main homepage — live post feed, filter bar, advanced search, campus resource links |
| `create-post.html` | Post creation form — select Need or Offering, category, title, description, availability |
| `browse-needs.html` | Browse all posts — stub page, backend integration in progress |
